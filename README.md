# 📘 Aplicación del Algoritmo DBSCAN para la Detección de Anomalías en Conjuntos de Datos Ruidosos

## 🧠 Contexto de la investigación
Este proyecto surge de la necesidad de detectar anomalías en sistemas complejos donde los métodos tradicionales (como reglas predefinidas o K-Means) no son suficientes.  
La investigación se centra en el algoritmo **DBSCAN (Density-Based Spatial Clustering of Applications with Noise)**, un método de aprendizaje no supervisado capaz de identificar agrupamientos de formas arbitrarias y aislar puntos de ruido como posibles anomalías.

## 🎯 Enfoque principal
- **Fundamentos teóricos:** Se explican los parámetros críticos (\(\varepsilon\), MinPts), las métricas de distancia y la clasificación de puntos (núcleo, frontera, ruido).  
- **Metodología:** Se realizó una revisión sistemática de artículos recientes, categorizando aplicaciones de DBSCAN en sectores estratégicos.  
- **Aplicaciones prácticas:** El trabajo analiza cómo DBSCAN se aplica en:
  - Automotriz: diagnóstico OBD y seguridad vehicular.
  - Energía y gas: detección de fraude y fallos de sensores.
  - Agua: análisis multi-fuente con correlación de consumo.
  - Ciberseguridad: detección de intrusiones en redes ópticas y datasets IoT.
  - Big Data / IA Verde: reducción de datos y optimización energética.
  - Salud y biología: conectómica cerebral, genética y trazabilidad alimentaria.
  - Infraestructura: monitoreo estructural en puentes y túneles.

## 📊 Lo que se logró
- Se demostró que DBSCAN es robusto frente al ruido y flexible en la detección de patrones anómalos.  
- Se compararon variantes híbridas como **LOF-DBSCAN**, **KM-DBSCAN** y **kNN-DBSCAN**, mostrando mejoras en escenarios de densidad desigual y Big Data.  
- Se resaltó la importancia de la **IA Verde**, optimizando consumo energético y reduciendo la huella de carbono de los modelos.  
- Se integraron técnicas de **Explainable AI (XAI)** como SHAP para interpretar los resultados y entender qué variables generan anomalías.

## 📌 Proceso del proyecto

### 1. Inicio
El trabajo comenzó con la formulación de la pregunta de investigación y objetivos generales. A partir de allí se definió un plan de trabajo progresivo para explorar, evaluar, aplicar y validar el uso de DBSCAN en distintos contextos.

### 2. Búsqueda y ampliación de bibliografía
Se inició con un conjunto reducido de artículos y posteriormente se **extendió la bibliografía** para abarcar más sectores y aplicaciones.  
Este proceso permitió cubrir áreas como energía, agua, minería, ciberseguridad e industria, garantizando un respaldo académico sólido.

### 3. Revisión de metadatos
Los artículos fueron gestionados en **Mendeley**, donde se revisaron y corrigieron los metadatos incompletos (autores, DOI, páginas, ISSN, etc.).  
Este paso aseguró que las referencias estuvieran completas y listas para integrarse en el documento académico.

### 4. Integración en Overleaf
Con el archivo `.bib` corregido, se trabajó en **Overleaf** para:
- Insertar las referencias con formato académico.  
- Usar el estilo **IEEEtran** para mantener coherencia.  
- Redactar secciones con estructura clara: introducción, objetivos, metodología, resultados esperados y bibliografía.

### 5. Refinación de la pregunta y objetivos
La pregunta inicial fue ajustada para ser más específica y medible, incorporando términos como optimización de recursos y reducción de falsos positivos.  
Los objetivos se reformularon siguiendo criterios académicos, utilizando verbos más precisos y alineados con la pregunta.

### 6. Matriz de trazabilidad
Se elaboró una **matriz de trazabilidad** para identificar qué artículos eran más relevantes para cada objetivo del trabajo.  
Esto permitió establecer una relación clara entre la bibliografía y los componentes del proyecto, asegurando coherencia metodológica.

### 7. Actividades complementarias
Durante el proceso se trabajó en:
- Tablas comparativas para verificar coherencia entre pregunta y objetivos.  
- Preparación de diapositivas para la exposición, incluyendo los parámetros clave de DBSCAN (\(\varepsilon\) y MinPts).  
- Guiones de exposición para explicar cada diapo de manera clara y breve.
